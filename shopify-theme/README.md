# KapdeLatte Shopify Theme Starter

This folder contains the Shopify-first KapdeLatte theme starter.

## Included files

- `layout/theme.liquid` for the base page shell.
- `templates/index.json` for the homepage section order.
- `templates/product.json`, `templates/collection.json`, and `templates/cart.json` for core storefront pages.
- `sections/kapdelatte-hero.liquid` for the hero and mood cards.
- `sections/kapdelatte-featured-products.liquid` for collection-driven product cards.
- `sections/kapdelatte-brand-story.liquid` for brand story, testimonials, and newsletter.
- `sections/main-product.liquid`, `sections/main-collection.liquid`, and `sections/main-cart.liquid` for Shopify-native storefront pages.
- `assets/kapdelatte-theme.css` and `assets/kapdelatte-theme.js` for the shared presentation layer.

## How to use it

1. Create a new Shopify theme or duplicate an existing development theme.
2. Copy this folder's contents into the Shopify theme root.
3. In the theme editor, open the homepage and connect a collection to the featured products section.
4. Adjust copy, mood cards, and studio details through the section settings.

This is a Shopify theme starter focused on a polished homepage plus the core product, collection, and cart flow. Search, customer accounts, and additional page templates can be added next as the store structure is finalized.
