# The KapdeLatte

Preview and Shopify-first storefront work for The KapdeLatte, a rooted modern Indian clothing brand.

## What is here

- `index.html` for the standalone homepage preview.
- `our-thread.html` for the brand story page preview.
- `styles.css` and `script.js` for the shared front-end presentation and interactions.
- `shopify-theme/` for the Shopify theme starter and core storefront templates.

## Current direction

- Premium casual wear, modern kurtas, breezy cottons, and easy co-ords.
- Indian-rooted visual language with earthy accent colours and warm neutral bases.
- Shopify-friendly section and template structure for the eventual production storefront.

## Next steps

- Replace placeholder visuals with real brand photography.
- Move the approved preview direction fully into the Shopify theme.
- Connect products, collections, filters, and store policies in Shopify.
